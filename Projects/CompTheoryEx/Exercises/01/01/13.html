Chapter 1: #1.13

We sketched below the DFA recognizing the complement of the language, here is a simple explaination of why it works. When the DFA first encounters a $\texttt{1}$, there are two options for the next $\texttt{1}$ it will encounter. If there is an odd number of $\texttt{0}$'s in between, then the DFA will accept. If there is an even number of $\texttt{0}$'s, then the DFA will accept if it encounters another $\texttt{1}$ because either there are an odd number of symbols between this $\texttt{1}$ and the first one or there is an odd number of symbols between this $\texttt{1}$ and the second one.
