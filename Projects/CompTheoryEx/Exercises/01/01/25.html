Chapter 1: #1.25

A finite state transducer is a 5-tuple $(Q,\Sigma, \Gamma, \delta, q_0)$, where

  1. $Q$ is a finite set called the states,
  2. $\Sigma is a finite set called the input alphabet,
  3. $\Gamma$ is a finite set called the output alphabet,
  4. $\delta : Q \times \Sigma \rightarrow Q \times \Gamma$ is the transition function, and
  5. $q_0$ is the start state.

We give a formal definition of computation. A $\textsf{FST}$ $M = (Q,\Sigma, \Gamma, \delta, q_0)$ outputs $y=y_1\cdots y_n$ on input $x=x_1\cdots x_n$ if a sequence of states $r_0, \dots, r_n$ in $Q$ exists with two conditions:

  1. $r_0 = q_0$, and
  2. For $i = 0, \dots, n-1$, $\delta(r_i, x_{i+1}) = (r_{i+1}, y_{i+1})$.
