Observe from the table or from the mechanism of the factorial that $f_p(n) = f_p(n-1) + j$ where $j$ is the greatest integer such that $p^j \mid n$. For instance, because $2^3 \mid 8$, we have $f_2(8) = f_2(7)+3$. This gives the following formula. \begin{align*} f_p(n) &= \sum_{i=p}^n \begin{cases}1 & \mbox{ if } p \mid i \mbox{ and } p^2 \not\mid i\\\vdots \\j & \mbox{ if } p^j \mid i \mbox{ and } p^{j+1} \not\mid i \end{cases} \end{align*} Since $p^i \mid n$ implies $p^j \mid n$ for $j < i$, we can simplify the formula by considering the number of integer $k < n$ such that $p^i \mid k$ for any $i \in \mathbb{N}$. Pedantically, this is exactly the number of times $p^i$ goes in $n$. $$f_p(n) = \sum_{i=1}^{\infty} \left\lfloor \frac{n}{p^i} \right\rfloor$$
