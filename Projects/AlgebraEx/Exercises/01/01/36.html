Chapter 1: #1.1.36

From Proposition 2, we know that $\forall x,y \in G, xy = x \Leftrightarrow y = 1, xy = y \Leftrightarrow x = 1$. Since we know 1 is the indentity and it is unique, we can construct this table with possible values separated by commas.

$\star$ $1$ $a$ $b$ $c$
$1$ $1$ $a$ $b$ $c$
$a$ $a$ $1$,$b$,$c$ $1$,$c$ $1$,$b$
$b$ $b$ $1$,$c$ $1$,$a$,$c$ $1$,$a$
$c$ $c$ $1$,$b$ $1$,$a$ $1$,$a$,$b$

By 1.1.31, we know that $G$ has an element of order 2, without loss of generality, we assume it is $a$. Then, the $a$ column (and row) is completely determined because you cannot have 2 equal entries in the same column (or row) as stated by the cancellation laws. We can also remove some possibilities using the cancellation laws and we obtain this table.

$\star$ $1$ $a$ $b$ $c$
$1$ $1$ $a$ $b$ $c$
$a$ $a$ $1$ $c$ $b$
$b$ $b$ $c$ $1$,$a$ $1$,$a$
$c$ $c$ $b$ $1$,$a$ $1$,$a$

If $b^2 = a$ then $b^3 \neq 1$ which means $|b| = 4$, so this is not possible. We use the same argument for $c$ and end up with this table.

$\star$ $1$ $a$ $b$ $c$
$1$ $1$ $a$ $b$ $c$
$a$ $a$ $1$ $c$ $b$
$b$ $b$ $c$ $1$ $a$
$c$ $c$ $b$ $a$ $1$

Since this table is symmetric, $G$ is abelian.
