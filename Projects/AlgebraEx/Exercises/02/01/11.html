Chapter 2: #2.1.11

Denote the subsets $A \times 1$, $1 \times B$ and $\Delta$ respectively and recall that $(a,b)^{-1} = (a^{-1},b^{-1})$.

  1. This subset is not empty because it contains $(1,1)$. For any $a, a' \in A$, we have $(a,1)(a'^{-1},1) = (aa'^{-1}, 1) \in A \times 1$, so we conclude $A \times 1$ is a subsgroup.
  2. This subset is not empty because it contains $(1,1)$. For any $b, b' \in B$, we have $(1,b)(1,b'^{-1}) = (1, bb'^{-1}) \in 1 \times B$, so we conclude $1 \times B$ is a subsgroup.
  3. This subset is not empty because it contains $(1,1)$. For any $a,b \in A=B$, we have $(a,a)(b^{-1},b^{-1}) = (ab^{-1}, ab^{-1}) \in \Delta$, so we conclude $\Delta$ is a subgroup.
